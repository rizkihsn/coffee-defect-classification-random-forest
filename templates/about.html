About Project

Latar Belakang & Metodologi

Implementasi AI untuk klasifikasi cacat biji kopi guna menaikkan standar mutu industri agrikultur

Studi Kasus & Tujuan

Biji kopi hijau berkualitas tinggi merupakan prasyarat utama untuk menghasilkan minuman kopi bercitarasa premium. Namun, proses klasifikasi cacat biji kopi secara manual memerlukan waktu lama dan sangat tergantung pada keahlian visual manusia yang subjektif.

Proyek ini dibuat untuk tugas kuliah Artificial Intelligence dengan studi kasus:
“Klasifikasi Cacat Biji Kopi Menggunakan Random Forest Berbasis Web Flask”

Tujuan utamanya adalah menyediakan sistem klasifikasi cacat biji kopi otomatis yang cepat, akurat, dan transparan, yang dapat mendeteksi hingga 17 jenis cacat biji kopi berdasarkan standar ekspor/SCA menggunakan dataset resmi Kaggle.

Tugas Akademik AI

Model Utama: Random Forest Classifier
Library: Scikit-Learn, OpenCV, Skimage
Preprocessing: Resize, Grayscale, Equalize, Normalization
Feature Extraction: HOG, LBP Texture, Color Hist, Edge Density, Morphological Features
1. Data Preprocessing

Gambar di-resize ke ukuran standar 128x128 piksel, dikonversi menjadi grayscale, dan dioptimasi kontrasnya menggunakan histogram equalization sebelum proses ekstraksi fitur dimulai.

2. Feature Extraction

Ekstraksi fitur gabungan (HOG untuk bentuk/kontur, LBP untuk tekstur permukaan, Canny edge untuk densitas garis, intensitas histogram warna, serta fitur morfologi area dan sirkularitas) menghasilkan representasi 451 dimensi fitur numerik per gambar.

3. Random Forest Training

Model dilatih menggunakan ensemble 200 Decision Trees dengan parameter min_samples_split=5 dan min_samples_leaf=2 untuk mencegah overfitting. Akurasi pengujian mencapai 95%+.

4. Real-time Deployment

Model diekspor ke format pickle dan diintegrasikan ke server Flask API, yang menerima request upload gambar dan merespon hasil klasifikasi beserta score keyakinan (confidence) kurang dari 100ms.
